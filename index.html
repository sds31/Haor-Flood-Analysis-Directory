📌 How to use this directory:
– Each card links to a complete interactive HTML map (hosted on GitHub Pages, or any static server).
– All maps open in a new browser tab. Use layer control (top‑right corner) to toggle flood scenarios.
– To add a new map: copy the <div class="map-card">...</div> block and edit the title, description, and link.
